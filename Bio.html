Week One.

The first week at Skill Distillery did not disappoint. We began by recapping content from the pre-work, familiarizing ourselves with Eclipse, and building simple projects e.g., a simple calculator. ... The day starts promptly at 8:30 a.m. and we can expect to have a pair programming project assigned each day around 5:00 p.m. We can also expect a project to be assigned each Friday evening, and due the following Monday morning.

On day four, we were tasked with the well-known programming challenges FizzBuzz and Fibonacci. The latter proved to be a logical challenge and valuable learning experience. Our weekend project was to design a program that sums an amount of money entered by the user, ask for a tendered amount, and then dispense the correct amount of change starting with the largest bills first. I found this project exciting. It allowed me to gain a better understanding of passing arguments to methods (and as we learned, by value, of course), using printf, hexadecimals, and conditional statements.
